# Banking Analysis Project

## 📌 Project Overview
This project focuses on analyzing banking loan data to generate insights on loan distribution, customer behavior, and financial risk.

## 🛠 Tools Used
- Excel / Power BI / Tableau
- SQL
- Python (optional)

## 📊 Key KPIs
- Total Loan Amount
- Total Funded Amount
- Loan Status Distribution
- Grade & Sub-grade wise Revolving Balance
- Default Rate
- Customer Segmentation

## 📈 Insights
- Majority loans fall under Grade B and C
- Higher default rate observed in lower grades
- Revolving balance varies significantly across sub-grades

## 📂 Dataset
Bank loan dataset containing customer financial and loan details.
